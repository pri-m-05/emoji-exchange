# Emoji Exchange Starter

A tiny React + TypeScript + Vite starter app for trading emoji collectibles.

## What it includes

- browse emoji listings
- filter by category
- create a listing
- send an exchange request
- accept or decline requests
- localStorage persistence so the app still feels alive after refresh

## Why this is a good test repo

It already builds, it has enough moving pieces to be interesting, and it is simple enough that an autonomous coding agent can add features on top of it.

Good follow-up agent tasks:
- add sort by rarity
- add dark/light theme toggle
- add a profile page
- add request history filters
- add animations or toast notifications

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Easy GitHub upload without VS Code

### Option A: GitHub website
1. Create a new empty GitHub repo.
2. Extract this project zip on your computer.
3. Open the repo on GitHub.
4. Click Add file.
5. Click Upload files.
6. Drag the extracted project folder contents into the browser.
7. Scroll down and click Commit changes.

### Option B: GitHub Desktop
1. Extract this project zip.
2. Open GitHub Desktop.
3. Click File → Add Local Repository.
4. Choose the extracted folder.
5. If it is not a git repo yet, let Desktop initialize it or create a new repo from the folder.
6. Click Publish repository.
